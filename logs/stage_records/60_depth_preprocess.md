# Stage60 DAVIS/YouTube-VOS Depth Preprocess

Date: 2026-06-27

## Goal

Generate StreamSplat-compatible DepthAnything V2 depth images for DAVIS and YouTube-VOS so the datasets can be used for large-scale anchor export and later adapter/selector training.

## Code

```text
scripts/run_stage60_depth_preprocess.py
```

The script enumerates dataset frames, runs StreamSplat `DepthAnythingWrapper`, normalizes each predicted depth map to 8-bit PNG, and writes a frame-level CSV plus summary JSON.

Supported layouts:

| Dataset | RGB input | Depth output |
|---|---|---|
| DAVIS | `JPEGImages/Full-Resolution/<sequence>/*.jpg` | `depthImages/Full-Resolution/<sequence>/*_pred.png` |
| YouTube-VOS | `train/valid/JPEGImages/<sequence>/*.jpg` | `train/valid/depthImages/<sequence>/*_pred.png` |
