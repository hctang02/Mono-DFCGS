# Stage Records

This folder records important process notes, outputs, results, and conclusions for each stage after the Stage56-68 planning update.

Each stage should get one Markdown file with:

- user request / stage objective
- implementation or commands
- output paths
- important metrics and all-frame PSNR tables when applicable
- conclusions
- caveats / next action
- commit hash after completion

The first file, `00_previous_rounds_summary.md`, summarizes the work completed before this folder was created.
